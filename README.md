# Project Gedeon

Project Gideon is a hobby embedded systems project focused on building a fully custom software stack from the ground up.
The goal is to implement a minimal HAL, RTOS, WiFi interface, and a simple TCP/IP stack, all without relying on high-level libraries.

The device acts as a WiFi hotspot with a lightweight captive portal that serves educational content, demonstrating how networking and embedded systems work internally.

This project is purely for learning and experimentation, with an emphasis on understanding low-level design, system architecture, and protocol implementation.
